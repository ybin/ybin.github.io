通过一个实例来展示如何在Java层使用Android binder进行跨进程通讯。

概况

我们要实现一个计算器的例子，服务端用Service实现，它可以计算加减乘除等各种运算，
客户端是一个Activity，通过绑定到服务端，调用服务端的接口进行计算，然后把计算结果
展示出来(TextView)。

对于我们这个例子来说，业务就是加减乘除各种计算，所以业务接口也就是计算器的各种接口。
