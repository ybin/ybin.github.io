基数排序(radix sort)

发表于 2014-11-14   |   分类于 算法

基数排序(radix sor)的C语言实现。

基数排序，是一种非比较性的正整数排序方法，其算法很简单：

  1. 找出所有数字的最大位数，如42就是2位
  2. 所有数字从最低有效位开始，每个有效位进行一次排序，数位较短的补0
  3. 排序完成之后就得到正确的排序结果

时间复杂度为: O(kN)，k为最大位数。
